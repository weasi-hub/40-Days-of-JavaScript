Loops and Iterations

Loop means doing the same task or execute a block of code repeatedly without writing the code for every repitition.

Loops are used when we want to automate repetitive tasks, like printing numbers, checking items in a list, or performing calculations repeatedly.

Types of Loop

  • for loop - best for fixed number of looping/iteration or when we know how many times we need to run a block of code.

  • while loop - A while loop runs as long as given condition is true. It's best when we don't know in advance how many iterations are needed.

  • do while loop - A do while loop ensures execute the code at least once before checking the conditions

  • Break and continue

    Break is for existing from the loop so that no more iteration.
    Continue is for skipping a particular iteration in a loop and go to the next one.
